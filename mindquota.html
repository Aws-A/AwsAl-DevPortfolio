A quiz app designed to share knowledge and encourage participation, featuring a simple, beautiful interface where users can also contribute to the app’s visuals

UX Research

We built our problem statement and UX hypothesis, which helped us
define our target persona as someone interested in expanding their
general knowledge and actively participating in
sharing his knowledge

Design Thinking

We created our user flow to map the user journey, guiding them from start to finish

I worked on two wireframes to define the elements we would use, then developed
them into prototype pages (A and B) to compare and decide which one
to move forward with

Database

We built the database and ERD, initially using MySQL to
store the data. Later, we transitioned to Firebase for data
storage. Initially, we created data tables for users,
quizzes, topics, questions, answers, and results.

Back-end

We initially developed the app using React with VS Code and deployed it
through GitHub, then transitioned to Flutter for mobile app development

Front-end

I used front-end technologies in VS Code, deployed the project on GitHub, and utilized
Android Studio to preview and interact with the app’s interface
